# 地支合局

## Part 1 业务流程

### 1.1 地支合局识别主流程

```mermaid
graph TD
    A["用户查看命盘的地支合局分析"] --> B["系统读取四柱地支数据"]
    B --> C["系统识别四柱间所有地支六合组合"]
    C --> D["系统识别四柱间所有地支三合局组合"]
    D --> E["系统对每组六合判断合局是否成立"]
    E --> F["系统对每组三合局判断合局是否成立"]
    F --> G["页面展示地支合局结果清单"]
```

### 1.2 六合与三合局成立条件判定流程

```mermaid
graph TD
    A["系统对六合与三合局逐一判断成立条件"] --> B{"六合组合是否紧邻?"}
    B -->|"是"| C{"合化之五行是否得令或得势?"}
    B -->|"否"| D["系统标注六合组合为遥合，力量减弱"]
    C -->|"是"| E["系统判定六合成立，标注合出五行"]
    C -->|"否"| F["系统标注六合合而不化"]
    A --> G{"三合局是否三字齐全?"}
    G -->|"三字齐全"| H{"三合局之五行是否得令或得势?"}
    G -->|"仅两字"| I["系统标注半合局，力量减弱"]
    H -->|"是"| J["系统判定三合局成立，标注合局五行"]
    H -->|"否"| K["系统标注三合局合而不化"]
    E --> L["页面展示合局成立结果与力量等级"]
    F --> L
    D --> L
    I --> L
    J --> L
    K --> L
```

## Part 2 关键页面功能列表

### 页面 / 功能 1: 地支合局结果页

- **URL / 路径（业务命名）**: 地支合局结果页
- **目标用户**: 命理学习者、命理从业者、普通用户
- **核心功能**:
  - 查看四柱间地支六合组合列表
  - 查看四柱间地支三合局组合列表
  - 查看每组六合的成立状态与力量等级
  - 查看三合局是否三字齐全或仅为半合局
  - 查看合局成立组合的合出五行属性
