# 神煞配置管理

## Part 1 业务流程

### 1.1 流派规则集管理流程

系统管理员维护不同流派的神煞规则集，每个规则集包含一组神煞定义及其识别条件。

```mermaid
graph TD
    A["管理员进入神煞配置管理页面"] --> B["系统展示当前已有的流派规则集列表"]
    B --> C["管理员选择操作"]
    C -->|新增流派规则集| D["管理员填写流派名称与说明"]
    D --> E["系统创建空白规则集"]
    E --> F["管理员向规则集内添加神煞定义"]
    C -->|编辑流派规则集| G["管理员选择目标流派规则集"]
    G --> H["系统展示该流派下的神煞定义列表"]
    H --> I["管理员修改、新增或删除神煞定义"]
    I --> J["系统校验神煞定义的完整性"]
    J --> K{"校验是否通过？"}
    K -->|是| L["系统保存修改"]
    K -->|否| M["系统提示校验不通过的原因"]
    M --> I
    C -->|删除流派规则集| N["管理员确认删除"]
    N --> O["系统移除该流派规则集"]
    L --> P["管理员设置某一流派规则集为默认生效规则"]
    P --> Q["系统更新默认流派标识"]
```

### 1.2 神煞定义增删改流程

管理员在某一流派规则集内对单条神煞定义进行增删改操作。

```mermaid
graph TD
    A1["管理员在流派规则集内点击新增神煞"] --> B1["管理员填写神煞名称"]
    B1 --> C1["管理员设定识别条件：基于天干、地支或天干地支组合的匹配规则"]
    C1 --> D1["管理员设定吉凶属性"]
    D1 --> E1["管理员填写神煞含义与判断要点说明"]
    E1 --> F1["系统校验识别条件格式与完整性"]
    F1 --> G1{"校验是否通过？"}
    G1 -->|是| H1["系统保存该神煞定义到当前流派规则集"]
    G1 -->|否| I1["系统提示条件格式错误或缺失字段"]
    I1 --> C1

    J1["管理员选择已有神煞定义进行编辑"] --> K1["管理员修改识别条件或吉凶属性或说明"]
    K1 --> F1

    L1["管理员选择已有神煞定义进行删除"] --> M1["系统提示确认删除"]
    M1 --> N1["系统从当前流派规则集中移除该神煞定义"]
```

### 1.3 神煞定义启停流程

管理员可暂时停用某条神煞定义而不删除，停用后该神煞不再参与识别。

```mermaid
graph TD
    A2["管理员在神煞定义列表中切换启用或停用状态"] --> B2{"当前是启用还是停用？"}
    B2 -->|停用转启用| C2["系统将神煞定义恢复为启用状态"]
    C2 --> D2["该神煞重新参与识别计算"]
    B2 -->|启用转停用| E2["系统将神煞定义标记为停用"]
    E2 --> F2["该神煞不再参与识别计算，命盘上不再标注"]
```

## Part 2 关键页面功能列表

### 页面 / 功能 1: 流派规则集列表页

- **URL / 路径（业务命名）**: 神煞流派管理
- **目标用户**: 系统管理员
- **核心功能**:
  - 展示所有流派规则集的名称与说明
  - 新增流派规则集
  - 删除流派规则集
  - 设置默认生效的流派规则集

### 页面 / 功能 2: 神煞定义列表页

- **URL / 路径（业务命名）**: 流派神煞定义列表
- **目标用户**: 系统管理员
- **核心功能**:
  - 展示当前流派规则集下所有神煞定义的名称与吉凶属性
  - 新增神煞定义
  - 编辑已有神煞定义的识别条件与含义说明
  - 删除神煞定义
  - 切换神煞定义的启用或停用状态
  - 按吉凶属性筛选神煞定义

### 页面 / 功能 3: 神煞定义编辑页

- **URL / 路径（业务命名）**: 神煞定义编辑
- **目标用户**: 系统管理员
- **核心功能**:
  - 填写神煞名称与别名
  - 设定识别条件（基于天干、地支或天干地支组合的匹配规则）
  - 设定吉凶属性（吉神或凶煞）
  - 填写神煞含义与判断要点说明
  - 校验识别条件格式的正确性与完整性
