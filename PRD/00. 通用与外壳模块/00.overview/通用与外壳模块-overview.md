# 通用与外壳模块

## 1. 简介

通用与外壳模块为产品提供应用级用户体验基础设施，涵盖三大跨模块能力：术语悬浮提示、多语言扩展基础和移动端适配。这些能力不针对特定业务功能，而是被所有领域模块（01–08）共同依赖——术语悬浮提示为各模块页面上出现的专业术语提供即查即释；多语言扩展基础为全站界面文字的本地化提供统一框架；移动端适配确保所有核心功能在手机浏览器上可用。

## 2. 关键概念

- **术语悬浮提示**：当界面中出现命理专业术语（如"用神""忌神""病机""十神"等）时，用户可通过鼠标悬浮（桌面端）或点击（移动端）查看该术语的通俗定义。术语定义与术语表（`0.common/glossary.md`）保持联动，术语表更新后提示内容同步更新。
- **多语言扩展基础**：为产品界面文字提供本地化框架。当前默认语言为简体中文（zh-CN），框架设计支持未来扩展其他语言。该基础仅处理界面层文字的切换与回退，不涉及领域内容（如命理论断文本）的翻译。
- **回退链**（Fallback Chain）：当用户选择的语言缺少某条界面文字的翻译时，系统按预设顺序逐级回退到上一级语言，最终回退到默认语言（zh-CN），确保界面不会出现空白文字。
- **移动端适配**：根据用户设备的屏幕尺寸与交互方式，自动切换布局与导航形态——桌面端使用侧边栏导航与鼠标悬浮交互，移动端使用底部标签栏导航与点击交互，确保核心排盘与论断功能在手机浏览器上完整可用。
- **断点策略**（Breakpoint Strategy）：响应式布局中区分桌面端与移动端的屏幕宽度阈值。屏幕宽度低于阈值时切换为移动端布局，高于阈值时保持桌面端布局。

## 3. 子模块及主要功能

| 子模块文件夹 | 主要功能 |
|-------------|---------|
| `01. 术语悬浮提示/` | 术语高亮标识、桌面端悬浮提示、移动端点击提示、术语表联动更新 |
| `02. 多语言扩展基础/` | 语言切换、语言回退展示、默认语言呈现、语言偏好保持、新增语言支持 |
| `03. 移动端适配/` | 响应式布局切换、移动端底部导航栏、移动端排盘输入适配、移动端论断阅读适配、触控交互适配 |
