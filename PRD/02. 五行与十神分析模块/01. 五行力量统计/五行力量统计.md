# 五行力量统计

## Part 1 业务流程

### 1.1 五行力量统计主流程

```mermaid
graph TD
    A["用户查看命盘的五行分析"] --> B["系统读取四柱天干地支与藏干数据"]
    B --> C["系统统计天干五行个数"]
    C --> D["系统统计地支本气五行个数"]
    D --> E["系统以月令为基准进行五行旺衰加权计算"]
    E --> F["系统识别五行缺失与五行过旺"]
    F --> G["页面展示五行力量分布、缺失与过旺标注"]
```

### 1.2 五行缺失与过旺识别流程

```mermaid
graph TD
    A["系统完成五行力量加权统计"] --> B{"某五行在天干与地支本气中均无出现?"}
    B -->|"是"| C["系统标注该行为五行缺失"]
    B -->|"否"| D{"某五行加权力量是否远超均值?"}
    D -->|"是"| E["系统标注该行为五行过旺"]
    D -->|"否"| F["系统标注该行为五行正常"]
    C --> G["页面展示五行缺失与过旺清单"]
    E --> G
    F --> G
```

## Part 2 关键页面功能列表

### 页面 / 功能 1: 五行力量分布页

- **URL / 路径（业务命名）**: 五行力量分布页
- **目标用户**: 命理学习者、命理从业者、普通用户
- **核心功能**:
  - 查看天干五行个数统计
  - 查看地支本气五行个数统计
  - 查看月令旺衰加权后的五行力量分布
  - 查看五行缺失标注
  - 查看五行过旺标注
