# 日主旺衰判定

## Part 1 业务流程

### 1.1 日主旺衰判定主流程

```mermaid
graph TD
    A["用户查看命盘的日主旺衰"] --> B["系统读取日主五行属性"]
    B --> C["系统判断日主是否得令（月令是否生扶日主）"]
    C --> D["系统判断日主是否得地（地支是否为日主之根）"]
    D --> E["系统判断日主是否得助（天干是否有同类或生扶日主）"]
    E --> F{"日主是否极旺且无可克泄?"}
    F -->|"是"| G["判定为从强"]
    F -->|"否"| H{"日主是否极弱且无生扶?"}
    H -->|"是"| I["判定为从弱"]
    H -->|"否"| J{"日主整体力量偏强?"}
    J -->|"是"| K["判定为身旺"]
    J -->|"否"| L["判定为身弱"]
    G --> M["页面展示日主旺衰结论与判定依据"]
    I --> M
    K --> M
    L --> M
```

## Part 2 关键页面功能列表

### 页面 / 功能 1: 日主旺衰判定页

- **URL / 路径（业务命名）**: 日主旺衰判定页
- **目标用户**: 命理学习者、命理从业者、普通用户
- **核心功能**:
  - 查看日主五行属性
  - 查看月令是否生扶日主（得令判断）
  - 查看日主在地支是否有根（得地判断）
  - 查看日主在天干是否有助（得助判断）
  - 查看日主旺衰结论（身旺、身弱、从强、从弱）
