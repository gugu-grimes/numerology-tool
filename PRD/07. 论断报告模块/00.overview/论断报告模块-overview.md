# 论断报告模块 概览

## 1. 模块定位

论断报告模块是本产品的核心交付层，将上游各分析模块（八字排盘与历法、五行与十神、合冲刑害、辨病与用神、神煞标注、大运流年）的分析结果综合为一份结构化的命理论断报告。报告以"辨病→论药→断吉凶"为主线串联八大章节，用户可选择生成全部或部分章节，并导出为 Markdown 或 PDF 格式。本模块本身不进行新的命理计算，而是汇聚和编排其他模块的输出，形成面向用户的最终交付物。

## 2. 关键概念

- **论断报告**：以"辨病→论药→断吉凶"为主线组织的结构化命理论断文档，是产品面向用户的最终交付物
- **八大章节**：报告的八个标准章节——基本命盘概览、命局诊断（识病）、用药方略（用神喜忌）、分维论断、岁运药效、神煞与特殊格局、合冲刑害详解、综合论断与建议
- **章节选择**：用户可在生成报告时勾选需要包含的章节，系统按主线顺序编排已选章节
- **病机依据链**：每条论断必须关联"所依据的病→所用之药→推导出的结论"，确保论断有据可依

## 3. 子模块清单

| 编号 | 子模块 | 文件夹 | 主要职责 |
|------|--------|--------|----------|
| 01 | 报告章节组织 | `01. 报告章节组织/` | 以辨病→论药→断吉凶为主线组织八大章节，用户可勾选生成全部或部分章节 |
| 02 | 报告导出 | `02. 报告导出/` | 将已生成的报告导出为 Markdown 或 PDF 格式文件 |
