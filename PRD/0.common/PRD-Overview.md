# 项目 PRD 概览

## 1. 系统概览

本产品是一款八字（四柱命理）解析与论断工具，面向命理爱好者、学习者及专业从业者。用户输入出生时间后，系统自动完成八字排盘，并依据传统命理学规则识别命局中的失衡与缺陷（"病"），再据病论断——辨病、论药、断吉凶，最终生成结构化的命理论断报告。

产品核心能力边界：支持公历与农历输入的排盘计算、五行旺衰与十神格局分析、合冲刑害关系识别、辨病识病与用神喜忌推导、大运流年岁运药效评估、神煞标注、命盘历史保存与双人合盘比较、结构化报告生成与导出。全部计算在本地完成，不依赖外部服务。

## 2. 关键概念

- **角色**：命理学习者（学习术语与论断逻辑）、命理从业者（快速排盘生成专业论断）、普通用户（了解自身运势）
- **辨病论断**：以"找病 -> 论药 -> 断吉凶"为主线的命理分析方法，所有论断均有病机依据
- **病机**：命局中失衡与缺陷的统称，包括日主过旺过弱、五行偏枯缺漏、十神交战、格局破败、合绊用神、寒暖湿燥失衡等类型（详见 glossary §2）
- **用神/喜忌体系**：治命局之病的药方，由用神、喜神、忌神、仇神、闲神五类构成（详见 glossary §1）
- **岁运药效**：将大运流年叠入命局，评估药方在岁运中的效力消长（详见 glossary §2）
- **命盘比较**：双人合盘时识别五行互补与冲克关系——此方之病是否为彼方之药
